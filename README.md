# Building a Retail Data Pipeline

This project demonstrates a complete Retail Data Pipeline using Python, SQL Server, Apache Airflow, and an interactive Dash dashboard.

## Technologies Used

- Python
- SQL Server
- Apache Airflow
- Pandas
- Plotly Dash
- Git & GitHub
